## 一、Lab5 实现进度

Lab5 实验要求完成 B+ 树索引的查找、插入与删除操作。

### 任务1：B+树查找操作的实现（5 个函数）

| 函数 | 所属类 | 功能 |
|------|--------|------|
| `lower_bound()` | IxNodeHandle | 在当前节点中二分查找第一个大于等于 target 的键位置 |
| `upper_bound()` | IxNodeHandle | 在当前节点中二分查找第一个大于 target 的键位置 |
| `leaf_lookup()` | IxNodeHandle | 在叶节点中查找等于 key 的键，返回对应的 Rid 指针 |
| `internal_lookup()` | IxNodeHandle | 在非叶节点中定位 key 所在的子节点页号 |
| `find_leaf_page()` | IxIndexHandle | 从根节点向下遍历，找到 key 所在的叶节点并返回 |

### 任务2：B+树插入操作的实现（5 个函数）

| 函数 | 所属类 | 功能 |
|------|--------|------|
| `insert_pairs()` | IxNodeHandle | 在当前节点的指定位置 pos 插入 n 个键值对 |
| `insert()` | IxNodeHandle | 在当前节点中插入 1 个键值对，保持键数组有序，重复 key 不插入 |
| `insert_entry()` | IxIndexHandle | B+树对外插入接口，将键值对插入叶节点，必要时调用 split 和 insert_into_parent |
| `split()` | IxIndexHandle | 分裂节点 node，更新叶节点兄弟指针或非叶节点父子关系 |
| `insert_into_parent()` | IxIndexHandle | 节点分裂后向父节点插入新节点信息，可能递归向上分裂 |

### 任务3：B+树删除操作的实现

根据实验指导书要求，实现 B+ 树的删除操作及相关辅助逻辑，包括节点的键值对移除、下溢处理（兄弟节点重分配或合并）、根节点调整等。具体涉及 `IxNodeHandle::erase_pair()`、`IxNodeHandle::remove()`、`IxIndexHandle::delete_entry()` 以及下溢处理相关的 `coalesce_or_redistribute()`、`redistribute()`、`adjust_root()` 等函数。

### 实现进度总结

Lab5 实验指导书要求的全部函数均已编写完成。
