## 一、项目概述

RucBase 是由中国人民大学数据库教学团队开发、配套教育部"101计划"核心教材《数据库管理系统原理与实现》建设的教学用数据库管理系统。本代码仓库面向哈尔滨工业大学数据库系统课程教学需要而定制。

- **编程语言**：C++17
- **构建工具**：CMake 3.16+
- **编译环境**：GCC 7.1+
- **依赖库**：flex、bison、readline

### 2.3 项目架构理解

```
Client → TCP Socket → rmdb.cpp (主入口)
                         ├── Parser (SQL解析)
                         ├── Analyzer (语义分析)
                         ├── Optimizer/Planner (查询优化)
                         ├── Portal (执行入口)
                         ├── Execution (查询执行)
                         ├── Index (B+树索引)
                         ├── Record (记录管理)
                         ├── Storage (缓冲池 + 磁盘管理)
                         ├── Transaction (事务管理 + 锁管理)
                         └── Recovery (日志与恢复)
```
