### 2.3 项目架构理解

```
Client → TCP Socket → rmdb.cpp (主入口)
                         ├── Parser (SQL解析)
                         ├── Analyzer (语义分析)
                         ├── Optimizer/Planner (查询优化)
                         ├── Portal (执行入口)
                         ├── Execution (查询执行)
                         ├── Index (B+树索引)
                         ├── Record (记录管理)
                         ├── Storage (缓冲池 + 磁盘管理)
                         ├── Transaction (事务管理 + 锁管理)
                         └── Recovery (日志与恢复)
```
